[h1]Psyern's Teleport Overhaul[/h1]

[img]https://i.imgur.com/SGMylzO.gif[/img]

[b]A standalone teleport overhaul for DayZ with UI travel, world teleports, loading screens, effects and safer server-authoritative handling.[/b]

Built for server owners who want a more polished teleport experience with better visuals, cleaner configuration, and improved loading behavior.

[h1]100% FREE[/h1]

[h1]Features[/h1]
────────────────────────────────────────

[list]
[*][b]World Teleports[/b] – Trigger teleports through configured world objects and interaction points
[*][b]F2 Teleport Menu[/b] – Open a full teleport UI with destination cards, images, cooldowns and travel actions
[*][b]Economy Support[/b] – Use free travel, Hardline reputation checks, Hardline reputation cost or item-based payment
[*][b]Destination Cooldowns[/b] – Configure cooldowns per destination with live countdowns in the menu
[*][b]Player Markers[/b] – Optional temporary markers show where a player teleported from after GUI travel
[*][b]Teleport Effects[/b] – Includes teleport particles and sound effects for departure and arrival
[*][b]Loading Screen[/b] – Fullscreen loading screen with random backgrounds, progress bar, HUD hide and audio fade
[*][b]Loading Optimizations[/b] – Preload support for undergrounds, interiors and heavy destination areas for smoother travel
[*][b]Teleport Safety[/b] – Uses safer teleport correction and delayed ground recheck to reduce bad spawns and terrain issues
[*][b]Server Authoritative[/b] – Teleport flow stays server-controlled for more reliable multiplayer behavior
[*][b]DayZ Expansion Hardline Compatible[/b] – Supports Hardline reputation checks and reputation cost travel when Hardline is installed
[*][b]Restricted Zones[/b] – Supports prohibited teleport zones with radius and polygon checks
[*][b]Localization[/b] – Includes stringtable support for localized UI and warning messages
[*][b]Translated Documentation[/b] – Loading screen customization guide available in 11 languages
[*][b]Logging[/b] – Writes teleport activity to a dedicated server log file
[*][b]Auto Config Generation[/b] – Creates and updates JSON configs automatically on first start or version changes
[/list]

[h1]Configuration[/h1]
────────────────────────────────────────

[list]
[*][b]teleport_config.json[/b] – World teleport entries and interaction teleports
[*][b]teleport_menu_config.json[/b] – F2 menu destinations, prices, cooldowns, markers and images
[*][b]teleport_preload_config.json[/b] – Global and per-teleport preload object setup
[*][b]ProhibitedZones.json[/b] – Radius and polygon no-teleport zones
[*][b]teleport.log[/b] – Teleport activity logging in the profile folder
[/list]

[h1]Overhaul Information[/h1]
────────────────────────────────────────

This mod is an overhaul of the original mod:

[url=https://steamcommunity.com/sharedfiles/filedetails/?id=3405070168&searchtext=teleport]Original Teleport Mod[/url]

We refined and upgraded the original system, improved multiple gameplay and UI elements, and added loading-time optimizations for a smoother teleport experience.

If you already used the original version, you can keep using its existing [b].json[/b] configuration files.
